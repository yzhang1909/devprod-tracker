# Dev Productivity Tracker

Small CLI tool to log daily work and summarize hours by project.

Quickstart

Install/run:

```bash
python3 cli.py init
python3 cli.py log --project example --hours 2.5 --notes "work"
python3 cli.py list
python3 cli.py summary --date 2024-01-01
```

Run tests:

```bash
python3 -m unittest discover -v
```
